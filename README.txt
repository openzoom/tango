--------------------------------------------------------------------------------

  OpenZoom Tango
  Adobe Photoshop Zoomify templates powered by OpenZoom Nano

--------------------------------------------------------------------------------


  Installation
  ------------

  Copy the files in this folder into the following
  folder on your system:

  * Adobe Photoshop CS3
    Windows: \Adobe\Adobe Photoshop CS3 Presets\Zoomify\
    Mac OS:  /Adobe Photoshop CS3/Presets/Zoomify

  * Adobe Photoshop CS4
    Windows: \Adobe\Adobe Photoshop CS4 Presets\Zoomify\
    Mac OS:  /Adobe Photoshop CS4/Presets/Zoomify


  Instructions
  ------------
  Export your pictures using Photoshop with
  File > Export > Zoomify...

  Choose from one of the six OpenZoom templates,
  give your image a name and press OK.
  Enjoy!

  ----------------
  * Fixed a bug where TileGroup calculation was wrong and certain

    the Zoomify image pyramid.

  ------------------
  * Added 100ms delay for loading images locally that previously
    caused trouble in Firefox on Mac OS X.

  ----------------

  * Fixed bug where local testing was not possible without adjustment
    of the Flash Player Security settings. (-use-network=false)
